GitHub token の Update について

2021/10/20

Takuji Sano

Git Credential manager core

最初は、Linux 版をインストールして設定しており、動作していたように見えましたが、token の有効期限切れで更新しようとしたときに、情報が少なく困りました。
色々見ていると、WSL2 に Ubuntu 版を入れて使っている人は少数派だったようです。
そのため、下記を参考に、設定を無効にしました。

  • http://memomo2.blogspot.com/2018/12/gitsource-treeauthentication-failed.html

というわけで、情報が多い Windows 版をインストールして、WSL2 と連携させて運用する方法に切り替えました。

  • https://bitto.jp/posts/%E6%8A%80%E8%A1%93/WSL/wsl-git-credential/

トークンの更新について

token の有効期限が近づいてきたら github からメールが来ます。
このときに token を再発行して設定するわけですが、再設定するコマンドを探しても見つけられませんでした。
Windows の認証を一度削除するとうまくいくようですので、下記のサイトを参考に設定してみました。

  • https://qiita.com/k3ntar0/items/03be3a1396a77a0d3f26
  • https://webbibouroku.com/Blog/Article/git-logout-win

次に git にアクセスするとユーザ名とパスワードを聞かれるので、そのときに新しいトークンを設定すれば解決と。
