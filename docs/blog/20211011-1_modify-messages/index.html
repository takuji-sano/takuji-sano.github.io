テーマを改造してみる

2021/10/11

Takuji Sano

埋め込まれているメッセージを更新する

探し方

最初に表示される写真の中央にオーバーレイされている文字列 “Tella, A Hugo theme for Companies” を書き換えてみる。
下記コマンドで、記載されているファイルを探します。

$ grep -r "Tella, A Hugo theme for Companies." ./*
./themes/tella/exampleSite/data/slide.json:    "title": "Tella, A Hugo theme for Companies.",
./themes/tella/README.md:Tella, A Hugo theme for Companies.
./themes/tella/data/slide.json:    "title": "Tella, A Hugo theme for Companies.",

grep に recursive のオプションつけて、対象は全ファイルって感じです。
結果としては data/slide.json に書かれてるみたいですね。

Contact ページの廃止

次に「Contact」ページを見てみます。
layouts/shortcodes/form.html を見ると、getform.io というサービスに飛ばしてるようです。
これは、問い合わせフォームの内容を getform 上に蓄積および管理ができるサービスのようで、アカウント作る必要があります。
当面つかう予定はないので、メニューから消したいですね。
メニューの設定は config.toml に記載されているので、消しておきます。
ローカルサーバで動作確認してみると…消えてない… サーバ再起動してみたら、消せたわ。

しかし、アイコンのつもりで選んだ画像が、巨大に表示されるのね…
この辺は、また後日。
